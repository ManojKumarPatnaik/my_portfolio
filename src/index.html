Password Management Tool

As a developer, I have developed one of the Password Management Tool (PMT) application that allows users to store multiple account details along with an encrypted password. The development of the PMT application was an iterative process, starting with the console for input and Collections framework as the backend, and progressing to MySQL database and Hibernate ORM for CRUD operations. We then shifted to Spring Core for dependency injection, and replaced the DAO layer implementation using Hibernate with Spring Data JPA. We used Spring MVC with JSP for presentation, and ultimately migrated to Spring Boot for creating REST APIs. The PMT application is a highly secure and scalable tool that supports various interfaces like web, smart devices, browser plugins, and application plugins. It allows users to generate random secure passwords with user-defined character sets and lengths, view saved passwords securely, group password accounts, and perform CRUD operations on password accounts and groups. Throughout the development process, we used a variety of tools and technologies, including Postman, IntelliJ Idea, Eclipse, STS, Sonar Lint, Git, Spring Boot, Maven, Spring Security, and various frameworks such as Spring, Spring MVC, Spring Data JPA, Spring Rest, JPA/Hibernate, JSP, HTML, CSS, Javascript, Lombok, Data Validation, JUnit, and Mockito. PMT application offers a range of features to help users manage their passwords securely and efficiently. These features include generating random secure passwords, grouping password accounts, creating and reading passwords for specific accounts, and modifying or deleting password accounts and groups. With PMT, users can easily organize and access their passwords, ensuring that their sensitive information is always protected. Overall, the PMT application is a powerful and secure tool that showcases our expertise in software development and our commitment to using the latest technologies and best practices.
